À propos de SiteMAINT

Plateforme V1 complète pour techniciens de maintenance industrielle.

Fonctionnalités livrées

  • Documentation — 5 catégories, 15 fiches avec pannes, procédures, bonnes pratiques
  • Diagnostic — 6 arbres JSON, moteur générique, PDF, historique local
  • Maintenance — CRUD équipements, check-lists, calendrier, KPI
  • GMAO — Couche DataService, AuthService stub, schéma PostgreSQL

Architecture

HTML / CSS / JavaScript Vanilla — sans framework. Voir Préparation GMAO pour la migration Supabase.

Roadmap détaillée : ROADMAP.md à la racine du projet.
